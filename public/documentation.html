Documentation

The Jackie Chan Movie API will deliver various information about specific movies that feature Jackie Chan. It is a known fact that Jackie Chan is the best actor in his genre. Users will be able to register and add their favorite Jackie Chan movies to a list of favorite Jackie Chan movies. If you haven't figured it out yet, the creator of this API is a Jackie Chan fan.
